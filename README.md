# 🚦 Traffic Sign Detection using YOLO

## 📌 Overview

This project implements a **real-time traffic sign detection system** using the YOLO (You Only Look Once) object detection model. It captures live webcam input and detects traffic signs with bounding boxes, labels, and confidence scores.

---

## 🚀 Features

* 🎥 Real-time detection using webcam
* ⚡ Fast inference with YOLO
* 🧠 Pre-trained traffic sign detection model
* 📦 Bounding boxes with labels and confidence scores
* 🖥️ Easy-to-run notebook implementation

---

## 🛠️ Tech Stack

* **Python**
* **OpenCV**
* **Ultralytics YOLO**

## ▶️ How to Run

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/traffic-sign-detection-yolo.git
cd traffic-sign-detection-yolo
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the project

* Open the notebook (`traffic_sign_detection.ipynb`)
* Run all cells
* Press **q** to exit webcam
